# Mini Challenge 2

## Issue Management Complete

### Acceptance Criteria
1. Add Login, Signup, Change Password, etc...
2. Full CRUDS support for `Issue`.
3. A board view that categories issues based on their status (3 columns: to do, in progress and done).

## Bonus
1. Make it so that each team has their own board (or filter based on team).
2. Deploy your project to heroku.

## Note
This project is less prescriptive. You get to decide what is important; Remember, this is an issue management system used at a company, likely to protect sensitive project data.
